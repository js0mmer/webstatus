This is the web server directory. You can configure the default files to your liking and/or place any  other files in
here. Make sure to configure the port properly in port.txt. If you have a dedicated host, make sure the port you specify
is open. Typically dedicated hosts have a specific range of ports that are open that you can find in their
knowledgebase/FAQs.

For writing HTML files, the following placeholder strings are valid:
%playerCards%
%playerCount%
%playerName%
%maxPlayers%
